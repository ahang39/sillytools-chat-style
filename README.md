# ✍️ 聊天样式 — 字体大小终于和缩放解绑了！

> **只改聊天字体，不动其他任何东西。**

---

## 为什么要做这个？

酒馆自带的字体调整其实是**缩放**——字变大了，按钮也跟着变大，菜单也变大，侧边栏也变大。在手机上这简直是灾难：调大字看聊天舒服了，一打开预设或者世界书编辑器就满屏都是按钮，根本没法操作。

这个扩展把**聊天字体大小**单独拎出来，只影响聊天消息区域的文字。按钮、菜单、预设面板全部不受影响。

特别适合和我一样的**纯文字卡爱好者**——不需要花里胡哨的 UI，只要聊天的字看着舒服就行。

## 能调什么

- 📏 **字体大小**：0-24px 拉滑块就行，0 = 恢复默认
- 📐 **行高**：自动跟随字号（1.6 倍），不用手动调
- 📄 **段落间距**：也是自动适配的

设置在扩展菜单 → 「聊天显示」。

---

## 📦 安装

1. 把 `sillytools-chat-style` 文件夹放进 ST 的 `data/<用户名>/extensions/` 目录
2. 重启 SillyTavern
3. 在扩展菜单里找到「聊天显示」，拖滑块调整
